модуль читает папку in, выводит, в out
при этом он может печатать информацию, перенапрявляем в out.txt
таким образом, для запуска модуля, нам нужно получить доступ к его папке, так как у него есть уже стартовая точка


если модуль завершился с ошибкой, он мог удалить все файлы, необходимо иметь возможность восстановить их

macbook выдерживает примерно 500 параллельных тредов достойно

модуль - раннер модуля - тред с раннером модуля - менеджер тредов

у менеджера есть настройки для каждого модуля (периодичность, входная папка)
у него есть общий таймер, открыте методы (запуск, остановка тредов)
должны получать список выполняющихся тредов
напротив каждого модуля должно иметься состояние (выполняется, остановлен)
